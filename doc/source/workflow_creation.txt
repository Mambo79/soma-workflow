=====================
Workflow creation API
=====================

The workflows are created using the soma.workflow.client API, see :ref:`examples` 
for a quick start.

Eventually, the objective is also to generate workflows from pipeline softwares.
The automatic workflow generation is already available for some `BrainVISA 
<http://brainvisa.info/>`_  pipelines.
